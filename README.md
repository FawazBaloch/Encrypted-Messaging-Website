# End-to-End Encrypted Messaging Web Application

This is a final year cybersecurity project that implements a secure messaging
and file-sharing system using end-to-end encryption.

## Features
- Real-time encrypted messaging using websockets
- Secure file sharing
- Hybrid encryption (AES + RSA)
- Localhost deployment

## Technologies Used
- HTML, CSS, JavaScript
- Python
- SQLite
- AES (symmetric encryption)
- RSA (asymmetric encryption)

## Security Design
- AES is used to encrypt message content
- RSA is used for secure key exchange
- Encryption ensures confidentiality of messages and files

## How to Run
1. Clone or download the repository
2. Run the application on localhost
